Lesson 3: Webpages, Documents, and Structure

HTML Documents

Webpages can use HTML as the structure, CSS for style, and Javascript in order to have interactive components.

HTML is the widely used markup language for web pages. It provides the structure of text information in a document so that certain text will be labeled as headings or paragraphs. It will also allow for embedded images and other objects to be supplemented for text.

Avoiding Repetition

Programmers need to try and avoid repetition. When they don't, this may lead to copying and pasting certain style attributes into many HTML elements. Programmers need to be consistent so that every lesson will look the same.
